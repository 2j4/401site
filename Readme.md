# 401 Website

Maybe it is just a training. =P

Use and Modify the Hexo Navy theme. 

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/setsal/401site.git
$ cd 401sitet
$ npm install
```

Build:

``` bash
$ hexo generate
```

Run developement server:

``` bash
$ hexo server
```
